Why Computer-Based feels “hard” even if your English is okay

CBT te main enemy hocche pressure + screen behavior. Paper er moto “calm reading” hoy na. Screen e panic korlei answer chole jay.

  • Typing slow → spelling + speed loss
  • Scrolling / navigation confusion → time waste
  • Timer shock → wrong section e stuck hoy
  • Reading screen fatigue → focus drops faster
  • Writing workflow broken → planning vs typing mix-up

Typing: “English jani, but type korte giye brain off”

CBT e typing is a band factor. Jodi type korte giye slow hoy—answer late hoy, spelling hoy, confidence break hoy. Fix: consistent micro-drills.

What to train (LizOn style)

  • Common IELTS spelling sets (dates, places, academic words)
  • Numbers typing under pressure (13/30, 15/50 type confusion)
  • Fast correction without panic (Backspace discipline)
  • Short burst typing (30–60 sec) like exam stress

Fast rule

Typing speed er goal “fastest” na. Goal hocche stable. Stability = more correct answers.

Navigation: screen flow jodi slow hoy, score down hoy

CBT te “where am I?” confusion is deadly. You need muscle memory: scroll → locate → answer → move.

  • Scroll smoothly (panic scroll na)
  • Question jump discipline: next question e instantly
  • Don’t “hunt for meaning” — hunt for the answer location
  • Keep your eyes on task objective, not UI noise

CBT e score stable hoy — jokhon screen flow stable hoy.

Time control: paper mindset CBT te fail kore

CBT te timer dekhe anxiety baray. Solution is not “try harder”. Solution is fixed time blocks + skip discipline.

Simple block strategy

  • Reading: section-wise target time + hard stop
  • Writing Task 2: plan (5) → write (30) → refine (5)
  • Listening: no rewind mindset + recovery mindset

The brutal truth

“One question e beshi time” মানে apni 2–3 ta easy mark sacrifice korlen.

CBT Reading: screen fatigue + focus loss

CBT Reading e passage er layout different লাগে. Eye fatigue hoy, scanning slow hoy. Fix is screen scanning strategy.

  • Skim first: structure ধরেন (intro / examples / conclusion vibe)
  • Keyword chase na — meaning block chase koren
  • Don’t over-scroll. Locate with purpose.

CBT Listening: read-ahead while audio plays

CBT Listening e jodi screen reading slow hoy, answer miss hoy. You must train: read-ahead speed + prediction.

  • Read next question before audio reaches it
  • Predict answer type (name/number/place/time)
  • Typing discipline: short + correct (no overthinking)

CBT Writing: typing + structure together রাখতে হয়

Writing e most students fail kore because তারা plan korte giye type slow, abar type korte giye plan ভেঙে ফেলে.

  • Template-like skeleton: intro → 2 body → conclusion (Task 2)
  • One paragraph one purpose
  • Quick edit: 3 things only (grammar, linking, repetition)

Exam-day strategy: score is built in the first 5 minutes

Exam day te “confidence” আসতে হবে routine theke, motivation theke na. CBT te control মানে routine.

Before you start

  • Breathing 10 sec → calm hands
  • Remind: “I don’t need perfect. I need stable.”
  • Decide your skip rule আগে থেকেই

During the test

  • One miss ≠ panic (recovery discipline)
  • Timer check only at fixed points
  • Move forward mindset always
